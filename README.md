# ⏳ Age Calculator (PyQt6)

A sleek, modern Desktop Application built with Python that calculates precise chronological age. Designed with a dark material theme and robust error handling to provide an intuitive user experience.

<img width="791" height="491" alt="AgeCalculator" src="https://github.com/user-attachments/assets/f402346a-c2cf-4a7a-8157-dd2f0c2da5c0" />


## 🧠 Engineering Features

* **Precise Date Parsing:** Accurate calculation of age down to years, months, and days, handling complex edge cases (leap years, month carry-overs).
* **Modern UI/UX:** Styled using `qt_material` (Dark Teal theme) for a professional, eye-friendly aesthetic, departing from native legacy GUI looks.
* **Exception Handling:** Implements `try-except` blocks to catch `ValueError`, preventing application crashes upon invalid date formats or empty inputs.
* **Responsive Grid System:** Utilizes `QGridLayout` to ensure UI elements scale and align perfectly.

## 🛠️ Tech Stack

* **Language:** Python 3.12
* **GUI Framework:** PyQt6
* **Theming Engine:** qt-material
* **Core Libraries:** `datetime`, `sys`

## 🚀 How to Run Locally

**1. Clone the repository**
```bash
git clone https://github.com/Nayanxyz/Age-Calculator-PyQt6.git 
cd Age-Calculator-PyQt6
```
**2. Install Dependencies**
```bash
pip install PyQt6
```
**3. Execute the Application**
```bash
python Age-Calculator.py
```
